### Список задач

```bash
python3 tasks.py get
```

### Создать задачу

```bash
python3 tasks.py create --title="New task" --notes="description" --completed
```

### Обновить задачу

```bash
python3 tasks.py update -n 1 --title="New title" --notes="new description" --completed
```

### Удалить задачу

```bash
python3 tasks.py delete -n 1
```
